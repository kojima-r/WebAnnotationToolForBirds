
requirements
----------------

```
gem install sinatra
gem install haml
gem install wav-file
```

サーバの起動
----------------

```
sh server_up.sh
```

起動後は、例えばブラウザから以下のようなアドレスにアクセス
```
http://localhost:4567/
```

ラベルファイル形式
----------------
１行あたり１ラベル（１区間）のコンマ区切りcsv
一行のフォーマットは以下の順序

```内部セグメントID,ラベル,時刻,方向,分離時セグメントID,教師ラベル=1```

注意事項
- 各行は入力順
- x座標の単位は(sec)
- y座標は0-1で正規化
